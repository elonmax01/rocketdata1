37.50. sql_parts #

Таблица sql_parts содержит информацию о различных частях стандарта SQL, поддерживаемых PostgreSQL.

Таблица 37.48. Столбцы sql_parts

Тип столбца

Описание

feature_id character_data

Строка идентификатора, содержащая номер части

feature_name character_data

Описательное название части

is_supported yes_or_no

YES, если часть полностью поддерживается текущей версией PostgreSQL, либо NO в противном случае

is_verified_by character_data

Всегда NULL, так как группа разработки PostgreSQL не проводит формальное тестирование функционального соответствия

comments character_data

Необязательный комментарий о поддерживаемом состоянии части
