37.58. udt_privileges #

Представление udt_privileges описывает права USAGE, назначенные текущей активной роли или текущей активной ролью для пользовательских типов. Оно содержит отдельную строку для каждой комбинации типа, праводателя и правообладателя. Это представление показывает только составные типы (почему, говорится в Разделе 37.60); права, затрагивающие домены, описаны в Разделе 37.59.

Таблица 37.56. Столбцы udt_privileges

Тип столбца

Описание

grantor sql_identifier

Имя роли, давшей право (праводатель)

grantee sql_identifier

Имя роли, которой было дано право (правообладатель)

udt_catalog sql_identifier

Имя базы данных, содержащей тип (всегда текущая база)

udt_schema sql_identifier

Имя схемы, содержащей тип

udt_name sql_identifier

Имя типа

privilege_type character_data

Всегда TYPE USAGE

is_grantable yes_or_no

YES, если право может передаваться, или NO в противном случае
