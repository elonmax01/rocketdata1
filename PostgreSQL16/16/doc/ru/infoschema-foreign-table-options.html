37.30. foreign_table_options #

Представление foreign_table_options показывает все параметры, определённые для сторонних таблиц в текущей базе данных. В нём отражаются только те сторонние таблицы, к которым имеет доступ текущий пользователь (являясь их владельцем или имея некоторые права).

Таблица 37.28. Столбцы foreign_table_options

Тип столбца

Описание

foreign_table_catalog sql_identifier

Имя базы данных, содержащей стороннюю таблицу (всегда текущая база)

foreign_table_schema sql_identifier

Имя схемы, содержащей стороннюю таблицу

foreign_table_name sql_identifier

Имя сторонней таблицы

option_name sql_identifier

Имя параметра

option_value character_data

Значение параметра
