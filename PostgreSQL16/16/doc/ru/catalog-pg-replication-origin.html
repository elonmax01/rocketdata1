53.44. pg_replication_origin #

В каталоге pg_replication_origin содержатся все созданные источники репликации. Подробно источники репликации описаны в Главе 50.

В отличие от большинства системных каталогов, pg_replication_origin разделяется всеми базами данных кластера: есть только один экземпляр pg_replication_origin в кластере, а не отдельные в каждой базе.

Таблица 53.44. Столбцы pg_replication_origin

Тип столбца

Описание

roident oid

Уникальный в рамках кластера идентификатор для источника репликации. Не должен покидать пределы системы.

roname text

Определяемое пользователем внешнее имя источника репликации.
