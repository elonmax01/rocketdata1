54.21. pg_rules #

Представление pg_rules открывает доступ к полезной информации о правилах перезаписи запросов.

Таблица 54.21. Столбцы pg_rules

Тип столбца

Описание

schemaname name (ссылается на pg_namespace.nspname)

Имя схемы, содержащей таблицу

tablename name (ссылается на pg_class.relname)

Имя таблицы, с которой связано это правило

rulename name (ссылается на pg_rewrite.rulename)

Имя правила

definition text

Определение правила (реконструированная команда создания)


Из представления pg_rules исключены правила ON SELECT для представлений и материализованных представлений; их можно увидеть в pg_views и pg_matviews.
