SPI_cursor_find

SPI_cursor_find — найти существующий курсор по имени

Синтаксис

Portal SPI_cursor_find(const char * name)

Описание

SPI_cursor_find находит существующий портал по имени. В основном это полезно для разрешения имени курсора, возвращённого в текстовом виде какой-то другой функцией.

Аргументы

const char * name

имя портала

Возвращаемое значение

указатель на портал с заданным именем или NULL, если такой портал не найден

Замечания

Помните, что эта функция может возвращать объект Portal, который не имеет свойств, подобных курсору, например, она может не возвращать кортежи. Если просто передать указатель Portal другим функциям SPI, они могут защититься от таких случаев, но при непосредственной проверке Portal следует проявлять осторожность.
