37.43. routine_sequence_usage #

Представление routine_sequence_usage идентифицирует все последовательности, используемые функцией или процедурой, либо в теле SQL, либо в стандартных выражениях параметров. (Это представление поддерживается только для тел SQL без кавычек, но не для тел или функций на других языках.) Последовательность выводится только в случае, если она принадлежит текущей активной роли.
