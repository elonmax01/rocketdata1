J.4. Сборка документации с помощью Meson #

Предусмотрены два варианта сборки документации с помощью Meson. Перейдите в каталог build перед запуском одной из этих команд или добавьте к команде параметр -C build.

Чтобы собрать HTML-версию документации: 

build$ ninja docs

Для сборки документации во всех форматах: 

build$ ninja alldocs

В результате документация появляется в подкаталоге build/doc/src/sgml.
