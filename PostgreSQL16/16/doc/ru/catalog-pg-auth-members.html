53.9. pg_auth_members #

Каталог pg_auth_members представляет отношения членства между ролями. Допускается любая не зацикленная иерархия отношений.

Так как пользователи определяются глобально, pg_auth_members разделяется всеми базами данных кластера; есть только единственная копия pg_auth_members в кластере, а не отдельные в каждой базе данных.

Таблица 53.9. Столбцы pg_auth_members

Тип столбца

Описание

oid oid

Идентификатор строки

roleid oid (ссылается на pg_authid.oid)

Идентификатор роли, включающей другую

member oid (ссылается на pg_authid.oid)

Идентификатор роли, включаемой в роль roleid

grantor oid (ссылается на pg_authid.oid)

Идентификатор роли, разрешившей членство

admin_option bool

True, если member может разрешать членство в roleid другим ролям

inherit_option bool

True, если член роли наследует привилегии выданной роли

set_option bool

True, если член роли может выполнять SET ROLE для выданной роли
