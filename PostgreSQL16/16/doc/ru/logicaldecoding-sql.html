49.4. Интерфейс логического декодирования на уровне SQL #

Подробнее API уровня SQL для взаимодействия с механизмом логическим декодированием описан в Подразделе 9.27.6.

Синхронная репликация (см. Подраздел 27.2.8) поддерживается только для слотов репликации, которые используются через интерфейс потоковой репликации. Интерфейс функций и дополнительные, не системные интерфейсы не поддерживают синхронную репликацию.
