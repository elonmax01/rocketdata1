Глава 16. Установка из двоичных пакетов

PostgreSQL доступен в виде двоичных пакетов для самых распространённых операционных систем. Если эти пакеты есть в вашей системе, устанавливать PostgreSQL рекомендуется именно из них. Сборка из исходного кода (см. Главу 17) рекомендуется только для разработчиков PostgreSQL или расширений для него.

Актуальный список платформ, для которых предоставляются двоичные пакеты, можно найти в разделе загрузок на сайте PostgreSQL по адресу https://www.postgresql.org/download/. Там же приведены инструкции по установке на определённой платформе.
