53.3. pg_am #

В каталоге pg_am хранится информация о методах доступа отношений. Каждая строка в нём описывает один метод доступа, поддерживаемый системой. В настоящее время методы доступа задаются только для таблиц и индексов. Требования для реализации табличных и индексных методов доступа подробно рассматриваются в Главе 63 и Главе 64.

Таблица 53.3. Столбцы pg_am

Тип столбца

Описание

oid oid

Идентификатор строки

amname name

Имя метода доступа

amhandler regproc (ссылается на pg_proc.oid)

OID функции-обработчика, предоставляющей информацию о методе доступа

amtype char

t = таблица (включая материализованные представления), i = индекс.


Примечание

До PostgreSQL 9.6, в pg_am было много дополнительных столбцов, представляющих свойства индексных методов доступа. Теперь эти данные непосредственно видны только на уровне кода C. Однако, чтобы SQL-запросы всё же могли проверять свойства индексных методов, была введена функция pg_index_column_has_property() и ряд связанных функций; см. Таблицу 9.72.
