37.40. routine_column_usage #

Представление routine_column_usage идентифицирует все столбцы, используемые функцией или процедурой, либо в теле SQL, либо в стандартных выражениях параметров. (Это представление поддерживается только для тел SQL без кавычек, но не для тел или функций на других языках.) Столбец выводится только в том случае, если его таблица принадлежит текущей активной роли.

Таблица 37.38. Столбцы routine_column_usage

Тип столбца

Описание

specific_catalog sql_identifier

Имя базы данных, содержащей функцию (всегда текущая база)

specific_schema sql_identifier

Имя схемы, содержащей функцию

specific_name sql_identifier

«Однозначное имя» функции. Подробнее оно описано в Разделе 37.45.

Имя базы данных, содержащей функцию (всегда текущая база)

routine_schema sql_identifier

Имя схемы, содержащей функцию

routine_name sql_identifier

Имя функции (может дублироваться в случае перегрузки)

Имя базы данных, содержащей таблицу, задействованную в функции (всегда текущая база)

Имя схемы, содержащей таблицу, задействованную в функции

table_name sql_identifier

Имя таблицы, задействованной в функции

Имя столбца, задействованного в функции
