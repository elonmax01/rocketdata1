17.2. Получение исходного кода #

Исходные коды PostgreSQL выпущенных версий можно загрузить из соответствующего раздела нашего сайта: https://www.postgresql.org/ftp/source/. Загрузите файл postgresql-версия.tar.gz или postgresql-версия.tar.bz2 интересующей вас версии и распакуйте его: 

tar xf postgresql-version.tar.bz2

При этом в текущем каталоге будет создан подкаталог postgresql-версия с исходными кодами PostgreSQL. Перейдите в этот подкаталог для продолжения процедуры установки.

В качестве альтернативы вы можете использовать систему контроля версий Git; за дополнительной информацией обратитесь к Разделу I.1.
