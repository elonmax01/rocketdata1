70.7. Примеры #

В базовый дистрибутив PostgreSQL включены классы операторов GIN, перечисленные ранее в Таблице 70.1. Также классы операторов GIN содержатся в следующих модулях contrib:

btree_gin

Функциональность B-дерева для различных типов данных

hstore

Модуль для хранения пар (ключ, значение)

intarray

Расширенная поддержка int[]

pg_trgm

Схожесть текста на основе статистики триграмм
