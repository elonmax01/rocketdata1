53.36. pg_parameter_acl #

В каталоге pg_parameter_acl представлены параметры конфигурации, для которых назначены права одной или нескольким ролям. Параметры, для которых действуют стандартные права, в этом каталоге отсутствуют.

В отличие от большинства системных каталогов, pg_parameter_acl разделяется всеми базами данных кластера: есть только один экземпляр pg_parameter_acl в кластере, а не отдельные для каждой базы данных.
