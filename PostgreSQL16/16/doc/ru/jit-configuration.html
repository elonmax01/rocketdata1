32.3. Конфигурация #

Переменная конфигурации jit определяет, возможно ли использование JIT-компиляции. Если она включена, переменные jit_above_cost, jit_inline_above_cost и jit_optimize_above_cost определяют, будет ли выполняться JIT-компиляция для запросов, и насколько ресурсоёмкой она может быть.

Параметр jit_provider определяет, какая реализация JIT-компиляции должна использоваться. Изменять его может потребоваться лишь в редких случаях. См. Подраздел 32.4.2.

Для целей отладки и разработки предназначен ряд дополнительных параметров конфигурации, описанных в Разделе 20.17.
