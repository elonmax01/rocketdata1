SPI_start_transaction

SPI_start_transaction — устаревшая функция

Синтаксис

void SPI_start_transaction(void)

Описание

Функция SPI_start_transaction не делает ничего, она сохранена только для совместимости с более ранними выпусками PostgreSQL. Ранее её надо было вызывать после SPI_commit или SPI_rollback, но сейчас эти функции начинают новую транзакцию автоматически.
