74.2. Транзакции и блокировки #

Идентификаторы выполняемых транзакций выводятся в представлении pg_locks, в столбцах virtualxid и transactionid. Для транзакций в режиме только чтения выводится только virtualxid, а в поле transactionid будет NULL, тогда как для транзакций чтения-записи выводятся оба идентификатора.

Некоторые типы блокировок ожидают virtualxid, другие типы ожидают transactionid. Информация о блокировках чтения и записи на уровне строк регистрируется непосредственно в заблокированных строках, её можно прочитать при помощи расширения pgrowlocks. Блокировки чтения на уровне строк также могут потребовать присвоения идентификаторов мультитранзакций (mxid; см. Подраздел 25.1.5.1).
